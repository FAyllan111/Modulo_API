Este repositório contém o projeto desenvolvido no módulo **"APIs com .NET"** do bootcamp **"Backend com .NET"** da DIO.

- **C#**  
- **.NET**  
- **ASP.NET Core**

- Manipulação de rotas e requisições HTTP.
